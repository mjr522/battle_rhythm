📋 CSV Schema Import Instructions

The Battle Rhythm CSV Importer maps column headers to task properties. To import tasks from Microsoft Excel, Google Sheets, or a text editor, ensure your CSV structure matches these rules:

Header Name Type / Required Accepted Values & Format Example
Title Required The name of the task card. (e.g. Faculty Evaluations)
Column Required The Area of Responsibility column name. (e.g. Ops, Curriculum)
POC Optional Role, specific position, or email POC. (e.g. Personnelist)
Recurrence Required Use either annual (recurring yearly) or one-off (occurs once).
Period Required Must be either: Early, Mid, or Late.
Months Required Full month names. Use semicolons for multiples (e.g. July;December;May)
Year Conditional Required ONLY if Recurrence is one-off. (e.g. 2026)
Trigger Optional Brief timeframe explanation label. (e.g. First week of October)
Notes Optional Notes, instructions, continuity details, templates.
StaffView Optional Set to TRUE (visible in Briefing mode) or FALSE (hidden in Briefing mode, POC view only). Defaults to TRUE.
Important Rules:
  • Spell the months fully with capitals (e.g., October, not Oct or 10).
  • If a column name in your CSV does not exist on the board, the board will **automatically create** the new column and assign the task to it.
  • Double-quotes are recommended around fields that contain commas or line-breaks (e.g., Notes).
